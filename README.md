## 💡 Troubleshooting Common Issues

If the software does not work as expected, check these common fixes first:

*   Game Not Found: Ensure the game is installed in a default directory. You can manually point the application to the game folder in the Settings tab.
*   Performance Drops: If your frame rate decreases, lower the upscaling quality or disable optional features like frame generation.
*   Driver Errors: Always verify that your display drivers are current. Outdated drivers often conflict with performance mods.
*   Access Denied: Run the application as an administrator if it fails to apply changes to game files. Right-click the app icon and select Run as administrator.
